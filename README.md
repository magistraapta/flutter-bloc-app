## Run Locally

Clone the project

```bash
  git clone https://github.com/magistraapta/flutter-bloc-app.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  flutter pub get
```

Run the app

```bash
  flutter run
```
